# Syuya Abe's Portfolio Website

Modern and responsive portfolio website showcasing my experience as a data scientist and software engineer.

## Overview

This portfolio website was rebuilt from scratch with a focus on:
- **Modern Design**: Clean, minimalist design inspired by contemporary portfolio sites
- **Responsive**: Fully responsive design that works on all devices
- **Fast Performance**: Optimized for quick loading and smooth interactions
- **Accessibility**: Built with semantic HTML and WCAG considerations
